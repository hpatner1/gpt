# Hassan Spot Trading Risk Manager

Framework-free crypto spot trading risk manager built with:

- Core PHP
- MySQL
- HTML/CSS
- Vanilla JavaScript

## Setup
1. Import `database.sql` into MySQL.
2. Update DB credentials in `config.php`.
3. Upload files to your shared hosting public directory.
4. Open `index.php` in browser.

## Core features
- User authentication (register/login/logout)
- Risk calculator for spot trades
- Trade journal (CRUD, search, pagination)
- Statistics and monthly performance
- Risk alerts (3 consecutive losses, weekly drawdown >5%)

No Laravel, Composer, or Node.js dependencies are required.
